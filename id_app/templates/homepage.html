About the project

The ID Card Creator is a comprehensive web application developed using Python and the Django framework, tailored to meet the needs of educational institutions, particularly colleges, for managing and generating student ID cards. This system aims to digitize the traditional, manual process of creating ID cards, offering a more streamlined and efficient solution.

Key Features

  • User friendly.
  • Create cards for institutions as well as each personnel .
  • Create and make changes to each entries.
  • Select the type of templete to generate card.
  • Deletion of entries.

Future implementations

  • Card template customization
  • Search filter
  • Desktop application
